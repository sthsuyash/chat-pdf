# DocuLume Documentation

Welcome to the DocuLume documentation site (Zensical). Edit the Markdown files in this folder to author your docs.

## Getting started

- Edit `zensical.toml` and set `site_name` and `site_url`.
- Run locally: `zensical serve` (from `docs/zensical`)
- Build: `zensical build`

## Example page

### Installation

Install the project dependencies and run the service locally.

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install zensical
zensical serve
```
