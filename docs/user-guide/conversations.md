# Managing conversations

Purpose: explain conversation lifecycle, saving, and exporting conversations.

Key features

- Create and continue conversations tied to uploaded documents.
- Save or export conversation transcripts for audit or sharing.

Procedure

1. Start a new conversation from the Chat page.
2. Use the document selector to scope retrieval to specific documents.
3. Save the conversation via the UI "Save" action.

Validation

- Saved conversations must be retrievable from the Conversations index.
