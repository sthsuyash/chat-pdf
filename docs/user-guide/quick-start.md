# Quick start

Purpose: get a user from zero to a working conversation in minutes.

Prerequisites

- An account on the deployed DocuLume instance or a local dev instance reachable from your browser.

Steps

1. Open the landing site in your browser.
2. Sign in using the provided test account or your SSO flow.
3. Navigate to "Docs" and verify the index loads.
4. Upload a small PDF or text document via the "Upload" UI.
5. Open the Chat/Conversations page and ask a simple question about the uploaded document.

Validation

- The uploaded document appears in the document list.
- Chat returns context-aware answers referencing the uploaded content.

Troubleshooting

- If uploads fail, check network logs and file size limits in the UI.
