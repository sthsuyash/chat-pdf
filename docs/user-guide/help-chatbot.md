# Help chatbot

Purpose: explain the built-in help/chatbot and when to use it.

Behavior

- The help chatbot answers common UI and workflow questions and can point to docs pages.
- It uses the same retrieval+LLM pipeline but is scoped to product help content.

Usage

- Open "Help" and type a question about using the site or uploading documents.

Validation

- The bot should return actionable steps or direct links to documentation pages.
