# Asking questions

Purpose: guidance on phrasing queries to get useful answers from DocuLume.

Guidelines

- Provide concise, specific questions (for example: "Summarize section 2 of document X").
- Reference document titles or upload IDs when relevant.

Best practices

- Use follow-up clarifying prompts rather than very long single queries.
- If answer seems incomplete, ask for citations or the source document reference.

Validation

- Answers should include a short excerpt or link to the source document when applicable.
