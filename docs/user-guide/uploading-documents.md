# Uploading documents

Purpose: describe the supported file types, limits, and recommended metadata.

Supported types

- PDF, TXT, DOCX. Keep single-file size under 50 MB for stable uploads.

Procedure

1. Visit the Upload page in the Docs section.
2. Select files and optionally add a title and tags.
3. Confirm and wait for processing to complete.

Validation

- The document should appear in the document list with processed status.
- Search should return results that reference the new document within a short delay.

Troubleshooting

- For processing failures, check backend logs and worker status.
